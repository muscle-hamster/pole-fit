# Social Media Images

## OG Image (Open Graph)

Place your Open Graph image here for social media sharing.

**File:** `og-image.jpg`

### Requirements:
- **Recommended size:** 1200 x 630 pixels (1.91:1 aspect ratio)
- **Format:** JPG or PNG
- **File size:** Under 1MB for best performance
- **Content:** Should include:
  - Jordega branding/logo
  - Jordan Garbett's name (optional)
  - Professional pole dancing imagery
  - Text should be readable at small sizes

### Usage:
The image will automatically be used when the website is shared on:
- Facebook
- Twitter/X
- LinkedIn
- Other social media platforms

### Tips:
- Use high contrast for readability
- Keep important text in the center (may be cropped on some platforms)
- Match the Y2K black and white aesthetic
- Test the image at 1200x630px to ensure it looks good
