# Portfolio Images

Place your portfolio images in this directory.

## Supported Formats:
- JPG / JPEG
- PNG
- WebP
- GIF

## Recommended Specs:
- **File size:** Under 2MB per image for best performance
- **Resolution:** 1080x1080 or higher (square format works best)
- **Format:** JPG or WebP recommended for smaller file sizes

## Naming:
- Use descriptive filenames (will be used for alt text)
- Files are automatically sorted alphabetically
- Any image files in this directory will automatically appear in the portfolio section

## Example:
- `performance-1.jpg`
- `event-photo-2.png`
- `showgirl-costume.jpg`

All images will be automatically displayed in the portfolio grid on the homepage.
