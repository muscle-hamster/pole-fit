# Video Files

Place your pole dancing hero video here.

## File Requirements:
- **Filename**: `pole-dancing-hero.mp4` (or `.webm` for better compression)
- **Format**: MP4 (H.264) or WebM
- **Recommended specs**:
  - Resolution: 1920x1080 or higher
  - Duration: 10-30 seconds (will loop)
  - File size: Keep under 10MB for best performance
  - Codec: H.264 for MP4, VP9 for WebM

## How to add:
1. Place your video file in this directory as `pole-dancing-hero.mp4`
2. Optionally add a `.webm` version for better browser compatibility
3. The video will automatically play in the hero section

## Video Tips:
- Use a video with good contrast so text remains readable
- Consider a video that showcases pole dancing skills
- The video will be slightly desaturated and darkened for the Y2K aesthetic
- Ensure the video is muted (required for autoplay)
